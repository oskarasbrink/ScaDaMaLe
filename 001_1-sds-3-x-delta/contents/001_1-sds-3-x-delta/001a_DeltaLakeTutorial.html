Conditional update without overwrite

Lets demonstrate using smaller a smaller DataFrame than the last notebook for simplicity. Remember from the last notebook that overwrite mode will overwrite records in the Delta table. If the table previously contained a range of 0-5, all records will now be overwritten to numbers 5-10.

data = spark.range(5, 10)
data.write.format("delta").mode("overwrite").save("/tmp/delta/example-delta-table")
data.show()

Conditional overwrites will do just that, only overwrite when some condition is met. Let's try out some basic examples. Worth noting is how we now load the delta table into a variable instead of loading the contents into a DataFrame. Use toDF() to transform it to a DataFrame, and then show() to show the contents of it.

from delta.tables import *
from pyspark.sql.functions import *

deltaTable = DeltaTable.forPath(spark, "/tmp/delta/example-delta-table")

The following code will update every record with an even value and then add 100 to it.

deltaTable.update(
  condition = expr("id % 2 == 0"),
  set = { "id": expr("id + 100") })

deltaTable.toDF().show()

Let's delete every odd value as follows:

deltaTable.delete(condition = expr("id % 2 == 1"))
deltaTable.toDF().show()

for more thourough examples and use cases of this, see Table deletes, updates, and merges.

Convert Parquet tables to Delta tables

Parquet files are used in many architectures. Delta tables also consist of parquet files in addition to the delta log. To convert a parquet table to a delta table, do as follows:

#create parquet table
df = spark.range(0,10)
df.write.parquet("/tmp/delta/delta-parquet-example")

And make sure there's only parquet files:

dbutils.fs.ls("/tmp/delta/delta-parquet-example")

Convert the table as follows:

DeltaTable.convertToDelta(spark, "parquet.`/tmp/delta/delta-parquet-example`")

If the table now indeed is a Delta table, we should see the delta log in it:

dbutils.fs.ls("/tmp/delta/delta-parquet-example")

And as in the previous notebook, we can actually test if it is a Delta table:

DeltaTable.isDeltaTable(spark, "/tmp/delta/delta-parquet-example")

Write a stream of data to a table

Spark Structured Streaming can be very useful for streaming applications using Delta architecture. It is possible to write data to Delta tables using structured streaming. Even in high speed and performant streaming pipelines, the Delta Lake transaction log guarantees exactly-once processing, even when there are other records being written concurrently. As a default, streams run in append mode, described as only new records are written to tables, opposite of overwrite mode where all records are overwritten.

Docs:

  • https://docs.delta.io/latest/quick-start.html#write-a-stream-of-data-to-a-table

Documentation and guides on Spark Structured Streaming:

  • https://spark.apache.org/docs/latest/structured-streaming-programming-guide.html
  • Notebooks in the ScaDaMaLe module 000_4-sds-3-x-ss on structured streaming.

Write a stream of data to a Delta table

streamingDf = spark.readStream.format("rate").load()
stream = streamingDf.selectExpr("value as id").writeStream.format("delta").option("checkpointLocation", "/tmp/checkpoint").start("/tmp/delta/example-delta-table")
